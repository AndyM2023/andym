HTML

HTML (Hypertext Markup Language) es un lenguaje de marcado utilizado para crear y estructurar el contenido de una página web. Consiste en una serie de etiquetas o elementos que se utilizan para definir el significado y la estructura del contenido de la página, como encabezados, párrafos, listas, enlaces, imágenes y formularios.Cada elemento de HTML está encerrado entre etiquetas, que se definen con símbolos de mayor y menor que (< y >), y pueden incluir atributos adicionales dentro de estas etiquetas para controlar cómo se presenta el contenido en la página.

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para definir la apariencia y el diseño de una página web. Con CSS, puedes controlar la presentación de elementos HTML, como el color del texto, el fondo, la fuente, el tamaño, la disposición y la animación.El CSS funciona en conjunto con el HTML: puedes enlazar un archivo CSS externo con tu documento HTML, o también puedes incluirlo directamente en el archivo HTML con la etiqueta style.Para aplicar estilos a los elementos de HTML, se utilizan las reglas de estilo, que consisten en un selector y un bloque de declaración. El selector apunta a un elemento específico del HTML que se desea estilizar, mientras que el bloque de declaración contiene las propiedades CSS que se aplicarán a ese elemento.
